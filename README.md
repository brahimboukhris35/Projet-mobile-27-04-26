BBQuiz

Application de quiz développée avec Flutter et Firebase.

Lancer l'application

Prérequis :
- Flutter installé
- Un navigateur Chrome ou Edge

Installation :

-git clone https://github.com/brahimboukhris35/Projet-mobile-27-04-26.git

-cd Projet-mobile-27-04-26

-flutter pub get

-flutter run -d chrome
